Custom Post Types UI plugin config for the EOU website

Use the CPT UI plugin to import the definitions for the custom post types:

  • Project
  • People - use instead of user since it is likely many staff will not be editors for the website and thus won't have regular user accounts

And the custom taxonomy:

  • Role - relationship with the organisation and any special position - staff, trustee, volunteer, treasurer, coordinator
  • Team - what they're responsible for (team/department) - E3, Comms, Caseworker
